# yyx-wx-butt

> 微信与服务器对接入口。
>
> **GET/POST方法均设置前缀*/wx*。所以微信公众号后台配置时，只需要设置项目名+"*/wx/*"即可。**

## 说明

这个模块主要是为了与微信公众号进行对接，以及提供微信访问自己服务器的一个入口模块。

即在下图中填写的URL为：http://domain/project_name/wx

其中的Token就是需要自行配置的config_token:[说明文档](../../README.md)
